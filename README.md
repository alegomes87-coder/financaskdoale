# Finanças do Alê — APK Builder

## Como gerar o APK (5 minutos)

### Passo 1 — Criar conta no GitHub
1. Acesse https://github.com e clique em **Sign up**
2. Preencha e-mail, senha e username
3. Confirme o e-mail

### Passo 2 — Criar repositório
1. Clique no **+** no canto superior direito → **New repository**
2. Nome: `financas-do-ale`
3. Deixe **Public** marcado
4. Clique **Create repository**

### Passo 3 — Fazer upload dos arquivos
1. Na página do repositório clique em **uploading an existing file**
2. Arraste **todos os arquivos e pastas** desta pasta para a área de upload
3. Clique **Commit changes**

### Passo 4 — Aguardar compilação (~5 min)
1. Clique na aba **Actions** do repositório
2. Você verá o job **Build APK** rodando (bolinha amarela)
3. Aguarde virar verde ✅

### Passo 5 — Baixar o APK
1. Clique no job concluído
2. No final da página em **Artifacts** clique em **financas-do-ale-apk**
3. Baixe o ZIP, extraia e instale o `.apk` no celular

### Instalar no celular
1. Transfira o `.apk` para o celular (WhatsApp, Drive, cabo USB)
2. Configurações → Segurança → Ativar **"Instalar de fontes desconhecidas"**
3. Abra o `.apk` e instale
